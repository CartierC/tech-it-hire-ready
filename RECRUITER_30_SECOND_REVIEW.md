## What Each Repo Proves

### 1. aws-ec2-secure-deployment
**Proves:** I can deploy AWS infrastructure from scratch, apply IAM least privilege, configure VPCs and security groups, and document the result with verification evidence.

**Signal for:** Cloud support, AWS infrastructure, secure deployment patterns.

### 2. it-automation-python
**Proves:** I can write Python scripts that solve real operational problems — not tutorial exercises, but tools designed to remove manual work.

**Signal for:** IT automation, scripting, operational tooling, systems engineering.

### 3. networking-labs-cli
**Proves:** I understand networking at the command line — subnetting, DNS, routing concepts, and network diagnostics — and can apply it hands-on.

**Signal for:** Cloud networking, IT support, infrastructure troubleshooting.

### 4. linux-hardening-pack
**Proves:** I can harden a Linux host with documented, repeatable steps — SSH config, firewall rules, security baselines — with clear reasoning for each control.

**Signal for:** Linux systems work, security-aware operations, infrastructure engineering.

### 5. devops-cicd-starter
**Proves:** I can build and explain a CI/CD pipeline using GitHub Actions — linting, validation, artifact handling — and apply it to real projects.

**Signal for:** DevOps, platform engineering, modern engineering team fit.

### 6. aios
**Proves:** I can implement AI agent systems at the engineering level — not just prompt engineering, but system design and tool-use orchestration.

**Signal for:** AI-adjacent infrastructure, technical operations in AI teams, forward-looking platform work.
