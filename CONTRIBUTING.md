# Contributing

Thanks for your interest in improving this repository.

## Purpose

This repository is a recruiter-facing portfolio hub for cloud, infrastructure, security, and automation work. Contributions should improve clarity, correctness, validation, and technical credibility.

## What to Contribute

Helpful contributions include:

- documentation fixes
- typo corrections
- validation improvements
- workflow improvements
- better evidence formatting
- broken link fixes
- safer defaults in scripts

## Before You Contribute

Please make sure your change:

- is directly relevant to the repository purpose
- keeps examples simple and professional
- does not add secrets, credentials, or private infrastructure details
- does not introduce placeholder content disguised as completed work

## Suggested Process

1. Open an issue describing the improvement
2. Fork the repository
3. Create a focused branch
4. Make the smallest useful change
5. Submit a pull request with a clear explanation

## Style Expectations

- Prefer readable, production-minded documentation
- Keep naming clear and consistent
- Use explicit validation steps where possible
- Do not add fake screenshots, fake outputs, or fake evidence

## Review Standard

Changes are evaluated based on:

- technical accuracy
- professionalism
- clarity for recruiters and reviewers
- alignment with the portfolio mission
