# Code of Conduct

## Our Standard

This repository should remain professional, respectful, and useful to reviewers, contributors, and learners.

## Expected Behavior

Examples of behavior that contribute to a productive environment include:

- giving constructive feedback
- focusing on technical accuracy
- respecting different levels of experience
- keeping discussion professional and relevant
- reporting issues responsibly

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment or personal attacks
- discriminatory or abusive language
- spam or intentionally disruptive behavior
- posting sensitive information without permission
- misrepresenting fake work as real technical proof

## Enforcement

The repository owner may remove comments, issues, or contributions that violate these standards.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and related project interactions.
